# Agent Stale Static Data Dynamic Check

This example demonstrates how AI agents can make incorrect decisions when relying solely on outdated static documentation like `AGENTS.md`. It contrasts an agent that blindly trusts static information with one that validates it against a dynamic, real-time data source. The example highlights the importance of dynamic validation to ensure agents act on current and accurate information.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
